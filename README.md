## base de datos
para cambiar la base de datos modificar "DB", en archivo config.js
### `DB:"firebase" o "mongo"`
## Modo Cluster
para cambiar a modo Cluster modificar "CLUSTER" y poner "on", en archivo config.js
### `CLUSTER:"on" o "no"`
